Diseño de Sitios WEB


Configuración de cuenta en www.github.com Se debe ingresar al link github.com y diligenciar el respectivo formulario.
Instalación de cliente de Git y github Descargar instalador desde github desktop una vez descargado e instalado, se deben ingresar correo y contraseña de la cuenta, para la instalación de git se debe descargar el instalador desde git y realizar la respectiva instalación y configuración.
Creación y sincronización repositorio de prueba Desde la cuenta de GitHub, se debe acceder al tab de repositorios, y se debe seleccionar la opción new y realizar la respectiva configuración del repositorio.
Configuración y sincronización de archivos Desde el cliente de git o la interfaz de github, se debe realizar la configuración de la cuenta y usuario de GitHub, ya sea por consola o por la interfaz, y despues de configurar la dirección del repositorio, se debe agregar los cambios con el comando ‘git add .’ luego se debe crear un commit, el cual indicará un punto en las versiones del repositorio, para eso se debe utilizar el comando ‘git commit -m “commit_notes” ’ y para finalizar, y sincronizar los archivos en el repositorio remoto, se debe ejecutar el comando ‘git push origin master’ y con este comando se subirán todos los cambios al repositorio remoto.
